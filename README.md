# Power-BI-Superstore-Analysis
This project involves a comprehensive Power BI Dashboard developed to analyze the sales performance of a global retail store. The goal was to transform raw sales data into actionable insights for executive decision-making, focusing on profitability, regional growth, and product trends.
Data Transformation: Used Power Query to clean data, fix data types (Text to Decimal), and handle null values.
Advanced DAX: Created complex measures for Total Orders, Profit %, and Estimated Revenue using functions like SUMX, DIVIDE, and DISTINCTCOUNT.
Data Modeling: Established a star schema relationship between the main Sales table and a custom Calendar table.
UI/UX Design: Implemented a dark-themed layout with conditional formatting to highlight negative profit areas instantly.
